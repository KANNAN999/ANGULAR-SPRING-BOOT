Basic Info and History

KV ToDo application is a web-based task management application. It allows users to manage their tasks from computer. Kv ToDo was first launched as a preview with basic features in April 2019. Later more features were added including Task updation and deletion in June 2018. In June 2020, a major update to the app was unveiled, adopting a new user interface with a closer resemblance to Memo app. The name was also slightly updated by removing the hyphen from To-Do.
